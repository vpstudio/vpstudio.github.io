Conclusion
Our project is more software oriented, which grants us the ability to easily add features or modify existing functionalities upon further thinking and suggestions. The hardware of the project is very stable, which will help quickly create a working prototype. Another good thing is that the most difficult parts we need to tackle are only related to secondary wants and are not critical to the basic functionalities, which gives us more confidence in completing this project.
The end goal of the VPS is to create a more accessible, portable drum kit than the traditional drum set. Users are given an physically interactive experience, far more immersive than today's phone and tablet apps. We hope to see more inspired drummers in the near future!
